# XD Chat

**Tagline:** Live Chat Made Simple

---

# Project Overview

XD Chat ek self-hosted, enterprise-grade, multi-workspace live chat platform hai jise kisi bhi website me sirf ek script ke through integrate kiya ja sakta hai.

Is project ka goal sirf ek chat widget banana nahi, balki future-ready communication platform develop karna hai.

---

# Vision

Ek aisa live chat platform banana jo:

* Fast ho
* Secure ho
* Lightweight ho
* Multi-workspace support kare
* Future SaaS ban sake

# Development Rules

## Rule 01

Planning first, coding second.

---

## Rule 02

One file = One responsibility.

---

## Rule 03

No duplicate code.

---

## Rule 04

No hardcoded values.

Always use Design Tokens.

---

## Rule 05

Every CSS variable starts with:

```text
--xd-
```

---

## Rule 06

Every CSS class starts with:

```text
.xd-
```

---

## Rule 07

Use `require_once` for PHP includes.

---

## Rule 08

Every folder must contain:

```text
README.md
CHANGELOG.md
```

---

## Rule 09

Every file must contain:

* Project Information
* Version
* Module
* Status
* Author
* Creation Date

---

## Rule 10

Design decisions are locked before coding.

Workflow:

```text
Requirement
↓
Research
↓
Design
↓
Approval
↓
Lock
↓
Development
```
